# Про кодстайл

1) Названия переменных lowercase, сокращаем, где можно, пробел = _

``` c
int num_of_elems = 0; // Объявляем и присваиваем сразу
```

2) Весь код на английском, в том числе в принтах, чтобы не было проблем с нечитабельным выводом. Комментарии можно на русском

3) При ошибке выводите что-нибудь, кроме тихого return -1; (см. errors.md и vector/generic.c)

4) Если как-то модифицируете структуру (например, добавляете вспомогательные функции), то делайте это в соответствующем заголовочному файлу стиле
